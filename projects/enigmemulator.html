ENIGMEMULATOR 8BILLION is a tool for writing secret messages, inspired by the famous Engima code machine used during World War II. It generates ciphers using nine different parameters, offering over 8 billion possible encryption settings.

To write a coded message, start typing in the input field. Adjust "scrambler", "start position" and "step-rate" parameters to customise your cipher, and generate a key so you can decode your message again later.
